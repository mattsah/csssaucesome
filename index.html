About This Site

The tricks you see here are designed to keep markup simple and semantic, placing additional burden on CSS thereby exposing new tricks and techniques.

This site is a bit of a playground designed to showcase some interesting CSS "hacks" which should also be practically useful as the web progresses. The goal is not to avoid javascript at all costs, but to draw attention to the fact that CSS is an extremely powerful tool.

About The Author

I'm a web developer with iMarc, LLC living in San Jose, CA and working in the Santa Cruz office. While I focus mostly on backend work for my job, I'm often tasked with implementing difficult design elements. I frequently come at CSS and HTML from the perspective of someone who deals with data, so I am consistently looking for simplicity and semantics.

Part of the reason I love CSS is because it keeps HTML clean.

Warning: This website uses a super small AJAX library which only supports standard XMLHTTPRequest objects. This prevents Internet Explorer version 10 and under from viewing individual tricks although most (all?) of these tricks should work in modern versions of IE, including 10.
