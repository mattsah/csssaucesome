Lightbox

Pure CSS lightbox which maintains markup simplicity. A single wrapping lightbox div with the ability to easily customize close button style/placement, background color

Support Required

  • Pseudo-select (:target)
  • Properties (max-width / min-width)
Close

CSS

		
/*
 * Operation
 */
.lightbox {
	display: block;
	position: fixed;
	top: 0;
	left: 100%;             /* position the block itself off the screen to not cover the page */
	width: 100%;            /* width and height should be 100% to take up full view */
	height: 100%;           /* if left was not 100%, this would mask the page itself */
	padding: 0;
	margin: 0;
	text-align: center;     /* center the image itself */
}

.lightbox:before {
	display: inline-block;  /* inline block allows for vertical centering */
	vertical-align: middle; /* align this guy to the middle */
	height: 100%;
	content: '';
}

.thumbnail {
	max-width: 250px;
	max-height: 250px;
}

.lightbox  img,
.lightbox .close {
	display: none;
}

.lightbox img {
	max-height: 75%;
	max-width: 90%;
	padding: 300%;      /* this combined with equal negative margin creates filler padding */
	margin: -300%;      /* offsets the padding */
	margin-left: -500%; /* offset left more to compensate for lightbox position */
	vertical-align: middle;
}

.lightbox img:target {
	display: inline-block;
}

.lightbox img:target ~ a.close {
	display: block;
	position: absolute;
}

/*
 * Layout and Theme
 */
.lightbox img {
	background-color: rgba(0,0,0,.75);
}

.lightbox a.close {
	width: 60px;
	height: 60px;
	line-height: 60px;
	font-size: 40px;
	border: solid 3px #fff;
	background-color: #000;
	border-radius: 40px;
	overflow: hidden;
	top: 20px;
	left: -100px;
	text-transform: uppercase;
	text-decoration: none;
	color: #fff;
	box-shadow: 0 0 20px #000;
}

.lightbox a.close:before {
	display: block;
	content: '❌';
}
